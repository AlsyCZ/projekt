This is my school project. If you want to run this project you need to replace the folowing code below with your database connection in every file which connects to database.

$host = 'localhost';

$dbname = 'Project';

$user = 'postgres';

$password_db = 'XXXXXXXX';

Connection to database is in files: 
add_comment.php, hardware_edit.php, hardware_process.php, index.php, load_coments.php, load_data_table.php,
load_game_options.php, load_gamedata_table.php, load_image.php, login.php, registrace.php

Build with: Visual Studio Code, PostgreSQL database and XAMPP (Apache)
